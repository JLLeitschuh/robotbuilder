Compressor

What is it?

The compressor keeps the pneumatics of the system pressurized up until the digital switch detects that there is too much pressure.

The pneumatics module will run the compressor automatically as long as there is a solenoid constructed. The compressor class is only required to turn off the compressor when it would normally be running

Properties

Module ID
The CAN ID of the pneumatics module that the compressor is plugged into.
Module Type
The type of pneumatics module that the compressor is plugged into. CTREPCM for the Cross the Road Electronics Pneumatics Control Module and REVPH for the Rev Robotics Pneumatics Hub.
