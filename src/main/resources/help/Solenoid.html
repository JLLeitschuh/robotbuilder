Solenoid (PCM)

What is it?

A solenoid that is plugged into the pneumatics module. It controls a pneumatic cylinder to create linear motion.

Properties

Output Module (Solenoid)
The CAN ID of the pneumatics module that this solenoid is plugged into.
Output Channel (Solenoid)
The channel on the pneumatics module that this solenoid is plugged into.
Module Type
The type of pneumatics module that the solenoid is plugged into. CTREPCM for the Cross the Road Electronics Pneumatics Control Module and REVPH for the Rev Robotics Pneumatics Hub.
