Motor Controller Group

What is it?

Controller for combining multiple motor controllers so they act as one. Useful for when multiple motor controllers control a single mechanism.

Properties

MotorController1
The first motor controller to add to the Motor Controller Group Group.
MotorController2
The second motor controller to add to the Motor Controller Group Group.
MotorController3 (Optional)
The third motor controller to add to the Motor Controller Group Group.
MotorController4 (Optional)
The fourth motor controller to add to the Motor Controller Group Group.
