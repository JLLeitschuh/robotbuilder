Conditional Command

What is it?

A conditional command runs one of two commands. The command to run is chosen during initialization of the conditional command based on the condition parameter.

Properties

Requires
The subsystem that this command requires.
OnTrue Command
Command to run when condition is true
OnFalse Command
Command to run when condition is false
Parameters
Variables to be passed into the command. Data can be of types boolean, byte, int, long, double, or string.
Parameter presets
Preset values for command parameters that can be used when a command is invoked
Button on SmartDashboard
Whether or not to automatically place a button on the SmartDashboard.
Run When Disabled
Sets whether or not this Command should run when the robot is disabled.
Sets whether or not this Command should be added to Autonomous Chooser.
