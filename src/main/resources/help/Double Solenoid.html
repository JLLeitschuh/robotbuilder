Double Solenoid (PCM)

What is it?

A Double acting solenoid that is plugged into a pneumatics module. It controls a pneumatic cylinder to create linear motion.

Properties

Forward Module (Solenoid)
The CAN ID of the pneumatics module that this solenoid is plugged into.
Forward Channel (Solenoid)
The port on the pneumatics module that this solenoid output is plugged into.
Forward Module Type (Solenoid)
The type of pneumatics module that the double solenoid is plugged into. CTREPCM for the Cross the Road Electronics Pneumatics Control Module and REVPH for the Rev Robotics Pneumatics Hub.
Reverse Channel (Solenoid)
The port on the pneumatics module that this solenoid output is plugged into.
